# Hausgeist Roadmap

## v0.1 — Foundation (Current)

**Goal:** Prove the loop works — add tasks via chat, see them on a board.

- [ ] Notion database setup with unified data model
- [ ] OpenClaw skill for task management
  - Add task: "Add task for Ira: brush teeth"
  - Complete task: "Ira finished brush teeth"
  - List tasks: "What's left for today?"
- [ ] Basic rules engine (hardcoded)
- [ ] Weekly summary via WhatsApp

**Success criteria:** Family uses it for 1 week without friction.

---

## v0.2 — Inventory

**Goal:** Track kids' stuff lifecycle.

- [ ] Inventory commands in OpenClaw
  - "Add winter jacket size 104 for Ira"
  - "Ira outgrew her shoes"
  - "What do we need to buy?"
- [ ] Hand-me-down suggestions (Ira → Isha)
- [ ] Shopping list view in Notion

---

## v0.3 — Documents & Time

**Goal:** Never miss an expiry or renewal.

- [ ] Document tracking (passports, visas, vaccinations)
- [ ] Expiry warnings (90/60/30 days)
- [ ] Calendar integration (sync due dates)
- [ ] Recurring tasks (daily/weekly chores)

---

## v0.4 — Smart Suggestions

**Goal:** AI that actually helps without being asked.

- [ ] Weather API → seasonal clothing prompts
- [ ] Local events API → kids activities nearby
- [ ] Age milestones → development-appropriate suggestions
- [ ] Price tracking → deals on "to-buy" items

---

## v0.5 — Custom UI

**Goal:** Kid-friendly visual board beyond Notion.

- [ ] Web app with colorful Kanban
- [ ] Avatars for family members
- [ ] Completion animations (confetti, sounds)
- [ ] Tablet-friendly for kitchen/fridge display
- [ ] Sticker chart mode for young kids

---

## Future Ideas

- Multi-family support (grandparents, shared custody)
- Meal planning module
- Budget tracking per category
- Voice interface (Alexa/Google Home)
- Physical NFC tags (tap to complete task)
